Project 03

FocusFlow

Html CSS Javascript Web Audio API localStorage

FocusFlow is a Pomodoro-based study timer designed to help students manage their focus sessions effectively. Built with pure HTML, CSS, and JavaScript, it features a circular progress ring, task list, session history, daily stats tracking, and a Web Audio API chime — all without any frameworks or external dependencies. Data persists locally so nothing is lost on refresh.

Key Features

  • Pomodoro timer with Focus / Short Break / Long Break modes
  • Animated SVG progress ring
  • Task list with completion tracking
  • Daily session history and stats
  • Sound alert via Web Audio API (no audio files needed)
  • Dark / Light mode with persistent preferences
  • Fully responsive, no frameworks used

Project 04

AllowanceIQ

Html CSS Javascript localStorage

AllowanceIQ helps students manage their weekly allowance by logging daily expenses under four categories — Food, Transport, School, and Others. The app tracks remaining balance in real time, displays a budget progress bar that turns orange at 80% and red when over budget, and shows a category breakdown so students know exactly where their money is going. A weekly summary displays total spent, average daily spending, and the top spending category. All data persists in localStorage and automatically archives the previous week every Monday.

Key Features

  • Weekly allowance setup with quick-amount presets (₱300 – ₱1,500)
  • Real-time remaining balance with color-coded status
  • Budget progress bar — turns orange at 80%, red at 100%
  • Expense logger with 4 categories — Food, Transport, School, Others
  • Category breakdown with individual mini progress bars
  • Weekly summary — total spent, average per day, top category
  • Auto-archives previous week's data every Monday
  • Dark mode by default with light mode toggle
  • Fully responsive for mobile and desktop
↗ Live Demo GitHub →

Project 05

VitalCheck

Html CSS Javascript localStorage

VitalCheck calculates your Body Mass Index based on height, weight, age, and gender — supporting both metric and imperial units. Results are displayed on an animated color-coded gauge that moves a needle to your exact BMI position, with the result card changing color based on your category. The app covers all six WHO classifications, shows your ideal healthy weight range, and provides four personalized health tips per category. Past results can be saved and tracked in a history log that persists across sessions.

Key Features

  • BMI calculator with metric (kg/cm) and imperial (lbs/ft) unit support
  • Animated BMI gauge with color-coded needle
  • 6 WHO BMI classifications — Underweight to Obese Class III
  • Dynamic result card that changes color per category
  • Healthy weight range based on your height
  • 4 personalized health tips per BMI category
  • BMI reference table with your current category highlighted
  • History tracking — save and review past results
  • Dark / Light mode with saved preference
  • Fully responsive for mobile and desktop
